**Garage Management System**
A simple full-stack Garage Management System built using the MERN stack.
This application helps manage customers, vehicles, and service requests in a garage.

**Features**
1. User login using JWT authentication
2. Role-based access (Admin / Mechanic)
3. Add, view, update, and delete customers
4. Manage vehicles and service records
5. Dashboard with basic revenue statistics
6. Responsive UI using Material UI

**Technologies Used**

**1. Frontend**
React, TypeScript, Vite & Material UI

**2. Backend**
Node.js, Express.js, MongoDB

**Project Modules**
1. Authentication
2. Customer Management
3. Vehicle Management
4. Service Management
5. Dashboard
